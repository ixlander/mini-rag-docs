Authentication

Password authentication

Users authenticate using an email and password. Passwords are stored using a secure hashing algorithm.

Password reset flow

When a user requests a password reset, the system generates a one-time token. The token is sent to the user's registered email address.

Password reset tokens are valid for 15 minutes and can be used only once.

Account lockout

Accounts are temporarily locked after multiple failed authentication attempts. This prevents brute-force attacks.
